Connect Google Drive
Your FD & RD data will be stored as fdrd_data.json in your Google Drive. Works on iOS, Android, and Desktop.
1
Go to console.cloud.google.com → Create a project (or select one)
2
Enable Google Drive API: APIs & Services → Library → search "Drive API" → Enable
3
APIs & Services → Credentials → Create → OAuth 2.0 Client ID → Web application
4
Under Authorised JavaScript origins, add the URL where this file is hosted (or http://localhost for local testing)
5
Copy the Client ID and paste it below
ℹ️ iOS note: The sign-in popup requires Safari on iOS. If blocked, go to Settings → Safari → allow popups for this site. Your Client ID is stored locally and never leaves your device.
