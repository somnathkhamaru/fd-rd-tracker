Connect Google Drive
Your FD & RD data will be stored as fdrd_data.json in your Google Drive. Works on iOS, Android & Desktop.
1
Go to console.cloud.google.com → Create a project
2
Enable Google Drive API: APIs & Services → Library → search "Drive API" → Enable
3
APIs & Services → Credentials → Create → OAuth 2.0 Client ID → Web application
4
Under Authorised JavaScript origins add only your domain e.g. https://yourname.github.io (no path, no trailing slash)
5
OAuth consent screen → Test users → add your Gmail address
6
Copy the Client ID and paste below
ℹ️ iOS: If the sign-in popup is blocked, go to Settings → Safari → allow popups. Your Client ID is stored locally and never leaves your device.
